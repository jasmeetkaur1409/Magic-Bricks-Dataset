# MagicBricks Housing Price Analysis
Exploratory Data Analysis on housing listings from MagicBricks, exploring how location, size, and amenities drive property prices across Indian cities.

📌 Project Overview

This project analyzes a real estate dataset to uncover pricing trends and key drivers behind property values. The goal is to extract insights that would be useful for buyers, sellers, or analysts trying to understand the housing market.

❓ Key Questions Explored


Which cities/localities have the highest average property prices?
How does price vary with area (sq. ft.), number of bedrooms (BHK), and furnishing status?
What's the relationship between price and amenities (parking, security, etc.)?
Are there outliers or unusual pricing patterns worth flagging?


🛠️ Tools & Libraries


Python
Pandas & NumPy – data cleaning and manipulation
Matplotlib & Seaborn – visualization
Jupyter Notebook – analysis environment


🔍 Approach


Data Cleaning – handled missing values, fixed inconsistent formatting, removed duplicates
Exploratory Analysis – univariate and bivariate analysis of price, area, BHK, and location
Visualization – distribution plots, correlation heatmaps, and city/locality-wise comparisons
Insights – summarized key patterns and pricing drivers
